Microorganisms · 미생물

미생물 · 원시생물

심해 생태계의 근간을 이루는 눈에 보이지 않는 생명들.

제노피오포라
지구 최대 단세포 생물로 크기 20cm 이상. 마리아나 해구에서 집중적으로 발견되며 해저 퇴적물을 먹습니다.
수심 6,000 ~ 11,000m
포람이니페라
탄산칼슘 껍데기를 가진 단세포 생물. 죽으면 해저에 쌓여 퇴적암의 주요 성분이 됩니다.
수심 0 ~ 11,000m
황화박테리아
빛 대신 황화수소로 에너지를 만드는 화학합성 생물. 심해 생태계 먹이사슬의 가장 밑바닥을 지탱합니다.
수심 2,000 ~ 4,000m
고세균
100°C 이상의 열수와 극압을 견디는 원시 미생물. 지구 생명의 기원을 연구하는 데 핵심적인 생물입니다.
수심 0 ~ 11,000m
심해 선충
해저 퇴적층 1㎡에 수백만 마리가 서식하는 실모양 벌레. 지구에서 가장 개체 수가 많은 동물입니다.
수심 0 ~ 11,000m
방산충
이산화규소로 만든 정교한 골격을 가진 단세포 생물. 결정처럼 아름다운 대칭 구조로 유명합니다.
수심 0 ~ 8,000m
규조류
유리 성분의 껍데기를 가진 단세포 광합성 생물. 지구 산소의 약 20%를 생산하는 핵심 생물입니다.
수심 0 ~ 200m
심해 바이러스
심해 미생물 군집의 수와 다양성을 조절합니다. 1ml 해수에 수억 개가 존재하며 물질 순환을 담당합니다.
수심 0 ~ 11,000m
섬모충
수천 개의 섬모로 이동하며 박테리아를 포식하는 원생생물. 극한 환경에서도 왕성한 번식력을 보입니다.
수심 0 ~ 9,000m
열수 고세균
열수분출공 300°C 이상 환경에서도 생존하는 극한 생물. 화성 등 외계 생명체 연구의 모델 생물입니다.
수심 2,000 ~ 4,000m
