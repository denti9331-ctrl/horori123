환형동물 · 극피동물

해저 생태계의 청소부이자 분해자 역할을 하는 심해 생물들.

관벌레
열수분출공에서 길이 2m 이상 자라며 화학합성으로 생존합니다. 소화 기관이 없어 공생 박테리아에 의존합니다.
수심 2,000 ~ 4,000m
폴리키트
수천 종이 심해 전역에 분포하는 다모류 털갯지렁이. 생물발광을 이용해 짝을 유인합니다.
수심 0 ~ 11,000m
좀비 벌레
고래 뼈 속 기름과 단백질을 분해하며 서식하는 오세닥스. 뿌리처럼 생긴 기관으로 뼈에 파고듭니다.
수심 50 ~ 4,000m
불꽃 벌레
몸에서 빛을 내며 분절된 강모로 이동합니다. 강모에 독 성분이 있어 포식자를 물리칩니다.
수심 0 ~ 3,000m
심해 해삼
해저 퇴적물을 먹으며 생태계 청소부 역할을 합니다. 위협 시 내장을 뿜어내 포식자를 교란합니다.
수심 0 ~ 10,000m
심해 불가사리
극도로 느린 신진대사로 극한 환경을 생존합니다. 팔 하나가 잘려도 재생할 수 있습니다.
수심 0 ~ 9,000m
거미불가사리
가느다란 팔로 해저를 기어다니며 먹이를 탐색합니다. 불가사리보다 팔이 훨씬 길고 유연합니다.
수심 0 ~ 6,860m
심해 성게
가시가 거의 없거나 길고 가늘게 변형된 형태. 오랜 수명과 느린 신진대사로 심해에 적응했습니다.
수심 0 ~ 7,000m
바다나리
수억 년간 형태가 변하지 않은 살아있는 화석. 꽃처럼 생긴 팔로 플랑크톤을 걸러 먹습니다.
수심 0 ~ 9,000m
바다백합
긴 자루로 해저에 고착하여 플랑크톤을 걸러 먹습니다. 4억 8천만 년 전 화석과 거의 동일한 형태입니다.
수심 100 ~ 9,000m
